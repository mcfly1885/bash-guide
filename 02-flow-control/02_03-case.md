# Case
 The `case` statement is useful when there are many conditions as it makes the code more readable and organized.

 Syntax
 ```bash
case $myvar in
    pattern1)
        # my code here
        ;;
    pattern2)
        # my code here
        ;;
    pattern3)
        #my code here
        ;;
        *)
        #my code here
esac

 ```
If myvar gets one of listed value the script executes the corresponding pattern.

Let's see an example

 ```bash
 #!/bin/bash
 
 color="red"

case $color in
    "yellow")
    echo "you have chosen yellow"
    ;;
    "orange")
    echo "you have chosen orange"
    ;;
    "red")
    echo "you have chosen red"
    ;;
    *)
    echo "the color $color is not available"
    ;;
esac
 ```
 The`;;` instruction is needed to stop the execution of code on that specific pattern and exit from `case` block.

 The keyword `*` is used to define everything else that is not specified in the pattern list. 

 In the example above, the colors to choose from are 3: `yellow`,`orange` and `red`.
 
  If the variable `color` equals, let's say, to `purple`, the script execute the code in `*` pattern and print the message "the color purple is not available".
